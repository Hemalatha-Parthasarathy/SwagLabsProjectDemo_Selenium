A robust test automation framework built using Selenium Webdriver, Java, Cucumber BDD, Page Object Model(POM),JUNIT, Maven.
This framework demonstrates industry-standard automation practices, including reusable page objects, BDD feature files, reporting, 
test data management, and scalable project structure.

Features:
- Selenium WebDriver with Java
- Cucumber BDD implementation
- Page Object Model (POM) design pattern
- Maven build management
- Cross-browser execution support
- Reusable utility classes
- Explicit wait implementation
- Test data management
- Detailed HTML reporting
- Easy scalability and maintenance
